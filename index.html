CSS শেখা (২৫টি প্রপার্টি)

colorরং

লেখার রং।

color: red;

backgroundপটভূমি

background: blue;

font-sizeআকার

font-size: 20px;

font-familyফন্ট

ফন্টের স্টাইল।

font-family: Arial;

text-alignস্থান

লেখার অবস্থান।

text-align: center;

widthপ্রস্থ

চওড়া মাপ।

width: 100px;

heightউচ্চতা

height: 50px;

marginবাইরে

margin: 10px;

paddingভিতরে

ভিতরের দূরত্ব।

padding: 10px;

borderবর্ডার

চারপাশে দাগ।

border: 1px solid;

border-radiusগোল

border-radius: 5px;

displayদৃশ্য

display: block;

font-weightমোটা

বোল্ড করা।

font-weight: bold;

opacityস্বচ্ছ

আবছা ভাব।

opacity: 0.5;

cursorমাউস

cursor: pointer;

box-shadowছায়া

overflow: hidden;
